NAACCR XML: Tumor.phase1NumberOfFractions

Description
Records the total number of fractions (treatment sessions) delivered to the patient in the first phase of radiation during the first course of treatment. This data item is required for CoC-accredited facilities as of 01/01/2018.
Rationale

Radiation therapy is delivered in one or more phases with each phase spread out over a number of fractions (treatment sessions). It is necessary to capture information describing the number of fraction(s) to evaluate patterns of radiation oncology care.
Codes
000 Radiation therapy was not administered to the patient.
001-998 Number of fractions administered to the patient during the first phase of radiation therapy.
999 Phase I Radiation therapy was administered, but the number of fractions is unknown; It is unknown whether radiation therapy was administered.
