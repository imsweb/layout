NAACCR XML: Tumor.measuredThickness

Description
Measured Thickness, or height, of a uveal melanoma, is a prognostic indicator for this tumor.
Rationale
Measured Thickness is listed as a Registry Data Collection Variable in AJCC. It was previously collected as Uveal Melanoma, CS SSF# 3.
