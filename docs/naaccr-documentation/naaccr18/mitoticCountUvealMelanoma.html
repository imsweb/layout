NAACCR XML: Tumor.mitoticCountUvealMelanoma

Description
Mitotic Count Uveal Melanoma, the number of mitoses per 40 high-power fields (HPF) based on pathological evaluation, is a prognostic factor for uveal melanoma.
Rationale
Mitotic Count Uveal Melanoma is listed as a Registry Data Collection Variable in AJCC. It was previously collected as Uveal Melanoma, CS SSF# 9.
