NAACCR XML: Tumor.ki67

Description
Ki-67 (MIB-1) is a marker of cell proliferation. A high value indicates a tumor that is proliferating more rapidly.
Rationale
Ki-67 (MIB-1) is a Registry Data Collection Variable in AJCC. It is a new data item for cases diagnosed 1/1/2018+.
