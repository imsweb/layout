1694 1 CoC 1763-1763

Description
Codes for the type of hormonal therapy given as part of the third course of treatment. Central registries currently collecting this data item should follow the 1998 ROADS Manual coding instructions. The codes are the same as those for Hormone Therapy, 1998 ROADS Manual, p. 238.

Note: This data item is no longer supported by CoC (as of January 1, 2003).
