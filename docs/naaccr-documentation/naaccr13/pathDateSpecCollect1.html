Path--Date Spec Collection
OBR-7 Observation Date/Time #00241 (HL7)		 7320 14 HL7 4580-4593

Description
Records the date and time of the specimen collection for the cancer being reported, not the date read or date the report was typed.

Rationale
Describes the origin of the pathology report contributing to this cancer abstract. The date that the specimen was collected is likely to correspond to the procedure date at the facility where a specimen was obtained.
