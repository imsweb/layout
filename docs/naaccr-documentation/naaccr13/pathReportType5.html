OBR-4 Universal Service ID #00238 (HL7)
Path--Report Type		 7484 2 HL7 5018-5019

Description
This field reflects the type of report transmitted to the cancer registry and may need to be classified at the central cancer registry.

This data item accommodates information for only one path report. Information in this data item should refer to the path report described in data items 7014, 7104, 7094, and 7194.

Rationale
Describes the origin of the fifth pathology report contributing to this cancer abstract. This variable is primarily used for administrative purposes at the cancer registry.
