Second Course of Therapy-Date Started (pre-96 CoC) 1660 8 CoC 1724-1731

Description
Date of initiation of second-course treatment. Central registries currently collecting this data item should follow the 1998 ROADS Manual coding instructions. See Chapter X for date format. Use SUBSQ RX 2NDCRS DATE FLAG [1661] if there is no appropriate or known date for this item.

Note: This data item is no longer supported by CoC (as of January 1, 2003).
