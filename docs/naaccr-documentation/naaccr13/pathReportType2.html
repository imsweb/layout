Path--Report Type
OBR-4 Universal Service ID #00238 (HL7)		 7481 2 HL7 4700-4701

Description
This field reflects the type of report transmitted to the cancer registry and may need to be classified at the central cancer registry.

This data item accommodates information for only one path report. If additional path reports were prepared, enter the path report type(s) in Path Report Type 3 through Path Report Type 5 [7482-7484]. Information in this data item should refer to the path report described in data items 7011, 7101, 7091, and 7191.

Rationale
Describes the origin of the second pathology report contributing to this cancer abstract. This variable is primarily used for administrative purposes at the cancer registry.
