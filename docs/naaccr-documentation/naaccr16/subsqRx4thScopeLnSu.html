1717 1 CoC 1997 5.1 1778 - 1778
Description
Codes for the type of surgery performed to remove regional lymph nodes as part of the fourth course of treatment. Central registries currently collecting this data item should follow the 1998 ROADS Manual coding instructions. The codes are the same as those for Scope of Regional Lymph Node Surgery, 1998 ROADS Manual, p. 192.
Note: This data item is no longer supported by CoC (as of January 1, 2003).
