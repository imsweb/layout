1699 2 CoC 1997 5.1 1759 - 1760
Description
Codes for the number of regional lymph nodes removed as part of the third course of treatment. Central registries currently collecting this data item should follow the 1998 ROADS Manual coding instructions. The codes are the same as those for Number of Regional Lymph Nodes Removed, 1998 ROADS Manual, p. 193.
Note: This data item is no longer supported by CoC (as of January 1, 2003).
