240 8 SEER/CoC

NAACCR XML: Patient.dateOfBirth

Alternate Names
   Birth Date(SEER/CoC/CCCR)
Description

Date of birth of the patient. See Data Descriptor Table for date format. If age at diagnosis and year of diagnosis are known, but year of birth is unknown, then year of birth should be calculated and so coded. Only the year should be entered, left-justified. Estimate date of birth when information is not available. It is better to estimate than to leave birth date unknown.
