NAACCR XML: Tumor.edpMdeLinkDate

Description

EDP MDE Link Date identifies the date when a breast, cervical, or colorectal cancer case in the registry database matched the same patient and tumor in CDC’s National Breast and Cervical Cancer Early Detection Program (NBCCEDP) data set, the CDC’s Colorectal Cancer Control Program (CRCCP) data set, or both, indicating that the cancer was identified through one or both early detection programs (EDPs). Use the EDP MDE Link Date variable to record results from the registry’s data linkage with the appropriate NBCCED/CRCCP program(s) in the registry’s state, territory, or jurisdiction.

Rationale

This data item records the date when a cancer was first linked with data from the early detection programs (EDPs) and allows analyses to evaluate trends, identify disparities, and evaluate program success. This linkage process can also identify cancers not reported or reported late to the central cancer registry.
