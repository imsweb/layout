NAACCR XML: Tumor.derivedPediatricStageGroup

Description

Derived Pediatric Stage Group is derived using the Pediatric Stage Data Items (Derived Pediatric T [1142], Derived Pediatric N [1143] and Derived Pediatric M [1144]) algorithm. Other data items may be included in the derivation process. Effective for cases diagnosed 1/1/2024 and forward.

Rationale

Derived Pediatric Stage Group can be used to evaluate disease spread at diagnosis, treatment patterns and outcomes over time.
