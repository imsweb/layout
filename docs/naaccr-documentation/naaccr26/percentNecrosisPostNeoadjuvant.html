NAACCR XML: Tumor.percentNecrosisPostNeoadjuvant

Description

Percent Necrosis Post Neoadjuvant is a prognostic factor for bone sarcomas.

Rationale

Percent Necrosis Post Neoadjuvant is a Registry Data Collection Variable for AJCC. It was previously collected as Bone, CS SSF# 3.
