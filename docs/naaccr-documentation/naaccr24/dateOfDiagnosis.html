NAACCR XML: Tumor.dateOfDiagnosis

Alternate Names
   Date of Initial Diagnosis (CoC)
Description

Date of initial diagnosis by a recognized medical practitioner for the tumor being reported whether clinically or microscopically confirmed. See Data Descriptor Table for date format.

For more discussion on determining date of diagnosis, consult the SEER Program Coding and Staging Manual or CoC STORE manual.
