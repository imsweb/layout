NAACCR XML: Tumor.invasionBeyondCapsule

Description

Invasion beyond capsule pertains to the pathologically confirmed invasion of the tumor beyond the fibrous capsule in which the kidney is enclosed.

Rationale

Invasion beyond capsule into specific tissues for Kidney is a Registry Data Collection Variable in AJCC. It was previously collected as Kidney, CS SSF #1.
