590 8 NAACCR

NAACCR XML: Tumor.dateOfInptAdm

Alternate Names
   Date of Inpatient Admission (CoC)
   Date of Inpatient Adm
Description

Date of the inpatient admission to the reporting facility for the most definitive surgery. In the absence of surgery, use date of inpatient admission for any other therapy. In the absence of therapy, use date of inpatient admission for diagnostic evaluation. See Data Descriptor Table for date format. Use DATE OF INPT ADM FLAG [591] if there is no appropriate or known date for this item.
