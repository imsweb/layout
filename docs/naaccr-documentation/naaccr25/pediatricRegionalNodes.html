NAACCR XML: Tumor.pediatricRegionalNodes

Description

Pediatric Regional Nodes is part of the Pediatric Staging data collection system and is used to classify the regional lymph nodes involved with cancer at the time of diagnosis. See also Pediatric Primary Tumor [1136] and Pediatric Mets [1138]. Effective for cases diagnosed 1/1/2024 and forward.

Rationale

Pediatric Regional Nodes is used to calculate Derived Pediatric N [1143] (when applicable). Derivation will occur at the level of the central registry.
