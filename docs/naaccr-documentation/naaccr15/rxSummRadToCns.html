Radiation to the Brain and/or Central Nervous System (SEER)
Radiation Therapy to CNS (CoC)
 1370 1 SEER/CoC 1581 - 1581
Description
For lung and leukemia cases only, codes for radiation given to the brain or central nervous system. Includes treatment given at all facilities as part of the first course. See Chapter V, Unresolved Issues, for more information.

Note: SEER does not collect this data item beginning with 1998 cases. They retain the codes for older cases in this field, and they have also recoded radiation coded here as radiation in RX Summ--Radiation [1360]. CoC does not collect this data item beginning with 1996 cases.
Codes

0 No radiation to the brain and/or central nervous system
1 Radiation
