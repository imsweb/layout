30 1 NAACCR 2-2

Description
A computer-generated code that best describes the type of registry generating the record; used when cases are pooled from multiple registries (a hospital-based registry reporting to a state should have a "3" in this field).

Rationale
Facilitates tracking of data sources when data from multiple registries are pooled.

Codes
1 Central registry (population-based)
2 Central registry or hospital consortium (not population-based)
3 Single hospital/freestanding center
