OBR-3 Filler Order Number #00217 (HL7)
Path Report Number		 7092 20 HL7 4772-4791

Description
Unique sequential number assigned by a laboratory to the third report for this case.

This item accommodates only one path report. When information is available for more than three path reports, enter the path report number(s) in Path Report No 4 through Path Report No 5 [7093-7094]. Information in this data item should refer to the path report described in data items 7012, 7102, 7192, and 7482.

Rationale
Describes the third pathology report contributing to this cancer abstract. The pathology report number provides a cross reference that identifies the specimen at the pathology facility. It may be useful for follow back with the pathology facility.
