NAACCR XML: Tumor.hcgPreOrchiectomyRange

Description

Human Chorionic Gonadotropin (hCG) Pre-orchiectomy Range identifies the range category of the highest hCG value measured prior to treatment. hCG is a serum tumor marker that is often elevated in patients with nonseminomatous germ cell tumors of the testis.

Rationale

hCG (Human Chorionic Gonadotropin) is a Registry Data Collection Variable in AJCC. hCG (Human Chorionic Gonadotropin) Pre-orchiectomy Range is used to assign the S Category Clinical and was previously collected as Testis CS SSF# 9.
