NAACCR XML: Tumor.microvascularDensity

Description

Microvascular Density (MVD), a quantitative measure of tumor vascularity, is a prognostic factor for uveal melanoma.

Rationale

Microvascular Density (MVD), is a Registry Data Collection Variable in AJCC. This data item was previously collected as Uveal Melanoma, CS SSF# 13.
