NAACCR XML: Tumor.prostatePathologicalExtension

Alternate Names
   Prostate, CS SSF# 3
   Prostate Pathological Extension
Description

EOD Prostate Pathologic Extension is used to assign pT category for prostate cancer based on radical prostatectomy specimens.

Rationale

EOD Prostate Pathologic Extension is used in EOD. It was previously collected as Prostate Pathological Extension, and Prostate, CS SSF# 3.
