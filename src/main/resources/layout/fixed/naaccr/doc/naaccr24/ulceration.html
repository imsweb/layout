NAACCR XML: Tumor.ulceration

Description

Ulceration, the absence of an intact epidermis overlying the primary melanoma based upon histopathological examination, is a prognostic factor for melanoma of the skin.

Rationale

Ulceration is a Registry Data Collection Variable in AJCC. It was previously collected as Melanoma Skin, CS SSF# 2.
