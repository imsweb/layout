470 1 NAACCR

NAACCR XML: Tumor.morphCodingSysCurrent

Description

Code that best describes how morphology is currently coded. If converted, this field shows the system it is converted to. An update or new Morphology Coding System code incorporates all previous coding updates.

2ICD-O, 1986 Field Trial
