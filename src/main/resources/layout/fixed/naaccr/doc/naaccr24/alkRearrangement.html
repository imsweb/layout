NAACCR XML: Tumor.alkRearrangement

Description

Testing for ALK rearrangement is performed for patients with advanced non-small cell lung cancer (NSCLC) to identify tumors which are sensitive to small-molecule ALK kinase inhibitors.

Rationale

ALK rearrangement is recommended by treatment guidelines for patients with advanced lung cancer as a prognostic marker and factor in determining appropriate therapy. It is a new data item for cases diagnosed 01/01/2021+.
