NAACCR XML: Tumor.dateCaseReportExported

Alternate Names
   Date Case Transmitted (pre-98 NAACCR)
Description

Date the reporting facility exports the electronic abstract to a file for transmission to the central registry. See Data Descriptor Table for date format. Standard edits check that no dates are later than the current date. Definitions may vary among registries and software providers.
