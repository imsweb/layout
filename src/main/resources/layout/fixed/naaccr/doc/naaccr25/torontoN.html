NAACCR XML: Tumor.torontoN

Description

This item stores the Toronto N value derived from coded fields the Toronto Stage algorithm (when applicable). Effective for cases diagnosed 1/1/2025 and forward.

Rationale

Toronto N can be used to evaluate disease spread at diagnosis, treatment patterns and outcomes over time.
