NAACCR XML: Tumor.subsqRx2ndCourseChemo

Description
Codes for the type of chemotherapy given as part of the second course of treatment. Central registries currently collecting this data item should follow the 1998 ROADS Manual coding instructions. The codes are the same as those for Chemotherapy, 1998 ROADS Manual, p. 228.
Note: This data item is no longer supported by CoC (as of January 1, 2003).
