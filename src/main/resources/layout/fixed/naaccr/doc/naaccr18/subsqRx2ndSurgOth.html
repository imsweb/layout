NAACCR XML: Tumor.subsqRx2ndSurgOth

Description
Codes for the type of surgery performed on tissue or organs other than the primary site and regional lymph nodes as part of the second course of treatment. Central registries currently collecting this data item should follow the 1998 ROADS Manual coding instructions. The codes are the same as those for Surgery of Other Regional Site(s), Distant Site(s) or Distant Lymph Node(s), 1998 ROADS Manual, p. 194.
Note: This data item is no longer supported by CoC (as of January 1, 2003).
