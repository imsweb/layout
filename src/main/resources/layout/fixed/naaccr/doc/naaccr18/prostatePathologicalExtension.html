NAACCR XML: Tumor.prostatePathologicalExtension

Description
Pathological extension is used to assign pT category for prostate cancer based on radical prostatectomy specimens.
Rationale
Pathological extension is used in EOD. It was previously collected as Prostate, CS SSF# 3.
