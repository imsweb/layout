NAACCR XML: Tumor.siteIcdO1

Alternate Names
   Primary Site (1973-91) (SEER)
Description
Area for retaining the ICD-O-1 primary site code entered before conversion to ICD-0-2. The item name includes years 1973-91. However, some states may have used the codes for cases before 1973.
Codes
For tumors diagnosed before 1992, contains the ICD-O-1 site code as originally coded, if available. Blank for tumors coded directly into ICD-O-2 (i.e., 1992 and later tumors).
