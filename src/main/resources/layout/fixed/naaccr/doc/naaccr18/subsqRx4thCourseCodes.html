NAACCR XML: Tumor.subsqRx4thCourseCodes

Description
The name for a group of subfields that describe the fourth course or set of subsequent therapy. As of January 1, 2003, CoC no longer support Subsequent Therapy data items.

Group names appear only in the data dictionary and Appendix E.
