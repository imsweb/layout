NAACCR XML: Tumor.hcgPreOrchiectomyLabValue

Description
hCG (Human Chorionic Gonadotropin) Pre-orchiectomy Lab Value refers to the hCG value measured prior to treatment. hCG is a serum tumor marker that is often elevated in patients with nonseminomatous germ cell tumors of the testis.
Rationale
hCG (Human Chorionic Gonadotropin) Pre-orchiectomy Lab Value is a Registry Data Collection Variable in AJCC. It was previously collected as Testis CS SSF# 8.
