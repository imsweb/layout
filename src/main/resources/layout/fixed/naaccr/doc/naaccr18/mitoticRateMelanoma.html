NAACCR XML: Tumor.mitoticRateMelanoma

Description
Mitotic Rate Melanoma, the number of mitoses per square millimeter based on pathological evaluation, is a prognostic factor for melanoma of the skin.
Rationale
Mitotic Rate Melanoma is a Registry Data Collection Variable in AJCC. It was previously collected as Melanoma Skin, CS SSF# 7.
