NAACCR XML: Tumor.subsqRx3rdCourseSurg

Description

Codes for the type of primary site surgery given as part of the third course of treatment. Central registries currently collecting this data item should follow the 1998 ROADS Manual coding instructions. The codes are the same as those for Surgery of Primary Site, 1998 ROADS Manual, p. 187.

Note: This data item is no longer supported by CoC (as of January 1, 2003).
