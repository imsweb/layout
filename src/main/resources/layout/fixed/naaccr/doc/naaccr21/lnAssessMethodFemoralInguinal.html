NAACCR XML: Tumor.lnAssessMethodFemoralInguinal

Description
This data item describes the method used to assess involvement of femoral-inguinal lymph nodes associated with certain female genital cancers.
Rationale
Method of assessment of regional nodal status is listed as a Registry Data Collection Variable in the AJCC GYN chapters. This data item was previously collected as Vulva, SSF# 15.
