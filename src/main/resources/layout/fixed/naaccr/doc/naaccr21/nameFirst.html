NAACCR XML: Patient.nameFirst

Alternate Names
   First Name (CoC)
Description
First name of the patient.
Note: The CoC STORE manual allows this field to be blank. If facilities with CoC-approved cancer programs submit blanks to the central registry, it is suggested that the central registry devise procedures for completing the last and first name with text, such as UNKNOWN, after verifying with the hospital that the field was left intentionally blank.

Note: See the most recent STORE for CoC allowable values.
