Coding System for Extent of Disease (SEER) 870 1 SEER 937-937

Description
Indicates the type of SEER EOD code applied to the tumor. Should be used whenever EOD coding is applied.

Rationale
Used in data editing and analysis.

Codes
0 2-Digit Nonspecific Extent of Disease (1973-82)
1 2-Digit Site-Specific Extent of Disease (1973-82)
2 13-Digit (expanded) Site-Specific Extent of Disease (1973-1982)
3 4-Digit Extent of Disease (1983-87)
4 10-Digit Extent of Disease, 1988 (1988-2003)
Blank Cases diagnosed 2004+; or the item is not collected
